### What I'm Building

**[Card Prices](https://card-prices.com)** — A free cross-platform pricing comparison tool for the collectible trading card market (Pokemon, Magic: The Gathering, Yu-Gi-Oh). Aggregates publicly available pricing data across global marketplaces so collectors and vendors can make informed decisions without paying for it.

**[Card Research](https://card-research.org)** — Academic research on pricing transparency in TCG secondary markets under market consolidation. IEEE paper in preparation.

### Current Focus

- Distributed data pipeline architecture across free-tier cloud infrastructure
- Cross-platform product identity resolution spanning 9 catalogs and multiple languages
- Cross-regional pricing analysis (US, Europe, Japan, South Korea, China)
- Mobile card scanning with multi-model OCR and visual identification

### Tech

Python, Node.js, Oracle Cloud, Firebase, distributed systems, data engineering, web scraping, cloud architecture on $0/month infrastructure
